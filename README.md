# dotfiles

These are my dotfiles. To use them clone the repo and use make. Make creates symbolic links to your home directory.

My files are heavily inspired by Jess Frazelle (https://github.com/jfrazelle)

Use this these commands to setup

### Install fonts
```console
sudo apt-get install fonts-inconsolata
sudo fc-cache -fv
```

### Clone repo and make links
```console
git clone --recursive https://github.com/gbbirkisson/dotfiles.git dotfiles
cd dotfiles
make
```
